**My Fitness App v1**



Core Requirements

* Sign-up/login
* Logging of calories
* Breakfast/lunch/dinner/snack interface for logging
* Food database (Including recipes and meals)
* Daily/weekly summary
* Calorie/macro goal tracking



Possible Features

* Workout tracking, including calories burned, logging of prs etc
* Weight goals (recommended calorie/macro intake)



Tech Stack

* JS ecosystem
* Vue 3 for front end
* MySQL for database management
* Node.js for backend
